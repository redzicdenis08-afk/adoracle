# Demo

This is the fastest way to show why Adoracle is worth starring. Use fictional example data only.

## Run it

```bash
npm install
npm run build
node dist/src/cli.js analyze examples/glowbrew_meta.json
node dist/src/cli.js report examples
```

## What to screenshot

Hook and angle detections, creative score, evidence spans, and advertiser rollups.

A good launch screenshot should show the command and the useful output in one image. Avoid giant terminal dumps.
